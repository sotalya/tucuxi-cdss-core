TucuCDSS HTML Response
***********************

This documentation provides an overview of the HTML report format that can be generated by our application. Our system supports the creation of HTML reports using a variety of basic templates, and it also allows for the creation of custom templates to meet specific needs.

.. contents:: Contents
	:depth: 1
	:backlinks: top
	:local:

Template Structure
==================

To create a custom template, store your HTML files in the following directory structure: html_template/{template_name}/index.html.  At report generation time,
the C++ binary uses the `Inja <https://github.com/pantor/inja>`__ template engine
to inject a JSON payload into the HTML template (replacing the ``{{ json_dump }}``
placeholder).  JavaScript code (``handle_input.js``, typically using jQuery) then
reads this JSON at page load and populates the DOM.


JSON Content
============

The application generates JSON content that is injected into the HTML page
and processed at runtime by ``handle_input.js``.  This JSON includes all the necessary data required for dynamic and interactive report generation. Below, you will find a detailed explanation of the JSON structure and its contents, which will help you to effectively utilize this data in your custom templates.
