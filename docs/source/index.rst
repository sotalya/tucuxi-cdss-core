Tucuxi-CDSS documentation
#########################

Tucuxi-CDSS is an open-source Clinical Decision Support System (|CDSS|) designed
to operationalize Model-Informed Precision Dosing (|MIPD|).
This documentation covers its architecture, configuration,
deployment, and operational use.
