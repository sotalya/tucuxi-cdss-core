User Guide
##########

This chapter provides the operational guidance needed to use
:program:`tucuxi_cdss_cli`.
It begins with the system's purpose and scope, describes the command-line
interface, walks through a typical integration workflow, and explains the output.
The closing sections cover exit codes, troubleshooting, known limitations, and
the build-and-test script.
