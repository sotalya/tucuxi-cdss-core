Glossary
========

.. glossary::
   :sorted:

   AUC
      Area Under the Curve.  Pharmacokinetic exposure metric expressing
      total drug exposure over a given time interval (typically one
      dosing cycle).

   CDSS
      Clinical Decision Support System. Software that combines
      a computational engine with data-validation and decision logic to
      assist clinicians in making evidence-based dosing decisions.

   CI
      Continuous Integration.

   CLI
      Command-Line Interface.

   Cmax
      Maximum drug concentration reached within a dosing cycle.

   Cmin
      Minimum (trough) drug concentration at the end of a dosing cycle.

   DOM
      Document Object Model.

   IEC 62304
      Medical device software: Software life cycle processes.

   IIV
      Inter-Individual Variability. Quantifies the spread of
      pharmacokinetic parameters across individuals in a population
      pharmacokinetic model.

   JSON
      JavaScript Object Notation.

   MAP
      Maximum A Posteriori. The Bayesian estimation method used by
      Tucuxi-Core to derive the most likely individual pharmacokinetic
      parameters given the population prior and observed drug
      concentrations.

   MIPD
      Model-Informed Precision Dosing. A dosing framework that
      integrates population pharmacokinetic models with Bayesian
      forecasting to individualize drug regimens.

   PD
      Pharmacodynamics. The study of the relationship between drug
      concentration at the site of action and the resulting
      pharmacological effect.

   PK
      Pharmacokinetics. The study of drug absorption, distribution,
      metabolism, and excretion over time.

   PK/PD
      Pharmacokinetics/Pharmacodynamics. The combined modelling
      framework linking dose to drug concentration (PK) and
      concentration to effect (PD).

   POC
      Point of Care. Refers to diagnostic or monitoring devices
      deployed at or near the patient's bedside, enabling real-time
      measurement without sending samples to a central laboratory.

   popPK
      Population Pharmacokinetics. Statistical modelling approach that
      characterizes the typical pharmacokinetic behaviour of a drug
      within a studied population while quantifying inter- and
      intra-individual variability.

   SDS
      Software Design Specification.

   SI
      Software Item.

   SRS
      Software Requirements Specification.

   TB
      Tuberculosis. The primary infectious disease context addressed
      by the TuberXpert project, which uses Tucuxi-CDSS for rifampicin
      dose optimization.

   TC
      Test Case.

   TDM
      Therapeutic Drug Monitoring. The clinical practice of measuring
      drug concentrations in patient samples to guide individualized
      dosing decisions.

   TQF
      Tucuxi Query Format. The XML-based file format used to supply
      patient data (dosing history, samples, covariates, therapeutic
      targets) to the :program:`tucuxi_cdss_cli` binary.

   TS
      Test Suite.

   V&V
      Verification and Validation.

   VVP
      Verification and Validation Plan.

   VVR
      Verification and Validation Report.

   XML
      Extensible Markup Language. The serialization format used for
      all Tucuxi-CDSS input queries, configuration files, language
      files, and output reports.
